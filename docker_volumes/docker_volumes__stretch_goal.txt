1. docker volume ls  (to see all existing volumes) 
2. docker volume rm [volume name] (started deleting them one by one while keeping my flaskapp container and mysql container running untill I got an error that one of the volumes can't be deleted because it's in use.) 
3. docker ps -a --filter volume=[volume name] ( to ensure which container is using this volume )
4. docker volume create [easy to remember volume name] ( created a new volume name for me to use with other container have presistant database )
5. docker run --rm -i -t -v [used volume name ]:/from -v [easy to remember volume name]:/to  alpine ash -c "cd /from ; cp -av . /to" ( to copy the content of the auto created volume to the volume I created )
6. docker stop mysql ( stop mysql container )
7. docker rm mysql ( delete the mysql container )
8. docker run -d -p 3306:3306 --name mysql --volume mysql-vol:/var/lib/mysql -e MYSQL_ROOT_PASSWORD=root mysql:5 ( create a new container with my volume )
9. docker network connect my-ne mysql ( connect the new container to the same network as flaskapp container )
10. docker is awesome ( this is not a command )
